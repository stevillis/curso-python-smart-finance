# Curso Python - Smart Finance

Este repositório contém os exercícios e materiais do curso "Python - Smart Finance". O objetivo do curso é apresentar os fundamentos de programação em Python com foco em aplicações financeiras e desenvolvimento de lógica de solução de problemas.

Link da sala de aula: [Google Classroom](https://classroom.google.com/c/ODY3MTk4NTMwMDA0?cjc=d4kuisam)

## Aula 1: Encontro 1 — Fundamentos + Introdução ao Projeto

- **Objetivo de Aprendizagem**: Ensinar a lógica, a sintaxe básica e preparar o ambiente de desenvolvimento.

Na primeira aula, vimos os conceitos básicos de programação em Python e como construir um pequeno programa interativo.

- **Tópicos para o Material**:
  - **Apresentação do SmartFinance AI**: Definição do escopo do projeto financeiro que será construído ao longo do curso.
  - **Configuração do Ambiente (Setup)**: Passo a passo para a instalação do Python (versão 3.10 ou superior) e do editor de código VS Code.
  - **Lógica de Programação do Zero**: Introdução teórica e prática ao pensamento algorítmico e resolução de problemas.
  - **Estrutura Básica do Python**: Explicação sobre variáveis, tipos de dados (strings, números e listas) e operações aritméticas fundamentais.
  - **Primeiros Scripts (Prática)**: Exercícios de criação de pequenos programas interativos utilizando entrada e saída de dados no terminal (Input/Output).

Principais tópicos abordados:

- `Hello world`: apresentação do primeiro programa em Python e execução básica.
- Entrada, Processamento e Saída: como ler dados do usuário, manipular valores e mostrar resultados.
- Variáveis: criação, nomes válidos, tipos simples e uso em cálculos.
- Tratamento de Exceções: uso de `try/except` para lidar com erros na entrada de dados.
- Estruturas Condicionais: uso de `if`, `elif` e `else` para tomar decisões no código.

## Aula 2

Na segunda aula, exploramos mais exemplos práticos de lógica em Python, estruturas de repetição e manipulação de dados para apoiar o aprendizado de programação financeira.

Principais tópicos abordados:

- Laços de repetição: uso de `for` e `while` para iterar sobre coleções e executar ações repetidas.
- Funções básicas: criação de funções para organizar o código e reutilizar lógica.
- Tipos de dados compostos: listas e dicionários para armazenar conjuntos de valores.
- Operações aritméticas e comparações: cálculos simples e condições mais complexas.

## Aula 3: Encontro 03 — Projeto Final e IA

Hoje é o dia de consolidarmos todo o conhecimento que construímos até aqui em um sistema financeiro real, funcional e inteligente. Vamos conectar a nossa lógica em Python e o Django para entregar a versão final do nosso SaaS!

Principais tópicos abordados:

- **🔐 Cadastro e Login**: Implementando o sistema de autenticação focado na segurança dos dados dos nossos usuários.
- **📊 Dashboard Financeiro**: Criando a interface visual completa com o resumo de saldos, gastos divididos por categoria e acompanhamento de metas.
- **👨‍👩‍👧 Gestão Familiar**: Desenvolvendo a estrutura lógica para suportar múltiplos perfis operando dentro da mesma aplicação.
- **🤖 Assistente IA**: A cereja do bolo! Vamos integrar nosso sistema com APIs de Inteligência Artificial para analisar automaticamente os gastos e gerar sugestões de economia.

💻 Preparem seus computadores, abram o VS Code e venham prontos para programar!
