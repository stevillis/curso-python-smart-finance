# Curso Python - Smart Finance

Este repositório contém os exercícios e materiais do curso "Python - Smart Finance". O objetivo do curso é apresentar os fundamentos de programação em Python com foco em aplicações financeiras e desenvolvimento de lógica de solução de problemas.

Link da sala de aula: [Google Classroom](https://classroom.google.com/c/ODY3MTk4NTMwMDA0?cjc=d4kuisam)

## Aula 1

Na primeira aula, vimos os conceitos básicos de programação em Python e como construir um pequeno programa interativo.

Principais tópicos abordados:

- `Hello world`: apresentação do primeiro programa em Python e execução básica.
- Entrada, Processamento e Saída: como ler dados do usuário, manipular valores e mostrar resultados.
- Variáveis: criação, nomes válidos, tipos simples e uso em cálculos.
- Tratamento de Exceções: uso de `try/except` para lidar com erros na entrada de dados.
- Estruturas Condicionais: uso de `if`, `elif` e `else` para tomar decisões no código.

## Estrutura do projeto

- `main.py`: arquivo principal do projeto.
- `aula-01/`: exercícios e exemplos da primeira aula.
